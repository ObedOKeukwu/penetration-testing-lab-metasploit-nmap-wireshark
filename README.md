# Penetration Testing Lab Project

## Overview

This project demonstrates a complete penetration testing workflow conducted within an authorized virtual lab environment. Activities included reconnaissance, network enumeration, vulnerability scanning, exploitation, post-exploitation, and web application security testing.

## Technologies Used

- Kali Linux
- Nmap
- Metasploit Framework
- Wireshark
- OWASP ZAP
- Netcat
- Windows XP
- Metasploitable 2

## Key Activities

### Reconnaissance
- Netcraft analysis
- WHOIS lookups
- DNS enumeration

### Network Enumeration
- Xmas scans
- SYN scans
- Protocol scans
- Firewall bypass testing

### Vulnerability Assessment
- Service identification
- Open port discovery
- Web application scanning

### Exploitation
- VSFTPD Backdoor Exploit
- MS08-067 NetAPI Exploit
- SSH access validation

### Post Exploitation
- Meterpreter session management
- System information gathering
- File transfer verification

## Skills Demonstrated

- Penetration Testing
- Vulnerability Assessment
- Network Security
- Threat Analysis
- Linux Administration
- Security Reconnaissance
- Incident Investigation
